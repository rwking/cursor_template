# Skill: MCP Server Development Patterns

## Description
This skill provides implementation patterns and templates for building robust, scalable Model Context Protocol (MCP) servers.

## Usage
Use this skill when:
- Bootstrapping a new MCP server project.
- Implementing complex tools with pagination, filtering, or heavy computations.
- Exposing resources with dynamic update notifications.
- Designing reusable prompts within the server context.
